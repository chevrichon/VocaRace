# VocaRace
## [Vocarace](https://chevrichon.github.io/VocaRace/) is a way to learn words in spanish while playing
The game is made by an ai right now but I'm coding right now an hand made version as my first coding project that actually look like something.
## Add words lists
To add a word list to the game just do it in the [Pull requests](https://github.com/chevrichon/VocaRace/pulls), for the format just look a the test.espagnol and lists.json files
